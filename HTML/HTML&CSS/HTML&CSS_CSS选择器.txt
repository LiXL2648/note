一、CSS简介：
	1. 层叠样式表
	2. CSS可以用来为网页创建样式表，通过样式表可以对网页进行装饰
	3. 所谓层叠，可以将整个网页想象成是一层一层的结构，层次高的将会覆盖层次低的
	4. 而CSS就可以分别为网页的各层次设置样式
二、CSS样式表：
		1. 内联样式表（行内样式表）：直接将样式写在元素（标签）的style属性当中。
			注：内联样式只对当前元素的内容起作用，不方便复用。
				内联样式属于结构与表现耦合，不方便后期的维护，不支持使用
		2. 内部样式表：样式表编写在style中，然后通过css选择器中指定元素，可以同时为这些元素一起设置样式，可以使样式的进一步复用。
			将样式编写到style标签中，也可以使表现和结构进一步分离，只支持当前页面推荐使用
		3. 外部样式表：可以将css样式表编写在外部的css文件中，然后通过link标签将外部的css文件引用到当前的页面中。
			<link rel="stylesheet" type="text/css" href="相对路径"/>
			将css样式统一编写到外部的样式表中，完全使结构和表现分离，然后通过link标签应用到页面中，实现css样式的复用
			加快用户的访问速度，提高用户的体验，最推荐使用
css注释：/*   */
HTML注释：<!---->
三、css的语法：选择器/声明块
	1. 选择器：通过选择器可以选中页面中指定的元素，并且将声明块中的样式应用到选择对应的元素上。

	2. 声明块：声明块紧跟在选择器的后面，使用一对{}括起来，声明块中实际上就是一组一组的明值对结构
			   在一个声明块中可以写多个声明，声明之间用;隔开，声明的样式名和样式值中间用:隔开
四、IDE
	1. IDE(集成开发工具)
	2. IDE拥有比纯文本编辑器更强大的提示功能，也是我们开发中用的比较多的工具

		ctrl+shift+r:复制一行

五、常用选择器：
	1. 标签选择器：通过标签选择器可以选择页面中的所有指定的元素
	2. id选择器（#id属性值）：通过元素id属性值选中唯一的一个元素，不推荐使用符合选择器
	3. 类选择器（.class属性值）：和id选择器类似，不过class属性可以重复,可以通过类选择器选择一组元素。
								 还可以为class属性设置多个值，中间用space隔开
	4. 选择器分组（并集选择器）：通过选择起器分组可以同时选择多个选择器对应的元素
				   语法：选择器1,,选择器2,选择器3,,......
	5. 通配选择器：可以选中页面中的所有元素。语法：*{}
	6. 符合选择器（交集选择器）：可以选中同时满足多个选择器元素，适合用类选择器
					语法：选择器1选择器2选中器3
	7. 后代选择器：选中指定元素的指定后代元素。语法：祖先选择器+space+后代选择器
	8. 子元素选择器：选中指定父元素的子元素。语法：父选择器>子选择器
六、元素之间的关系
	1. 父元素：直接包含子元素的元素
	2. 子元素：直接被父元素包含的元素
	3. 祖先元素：直接或者间接包含元素的元素，父元素也是祖先元素
	4. 后代元素：直接或者被包含的元素，子元素也是后代元素
	5. 兄弟元素：同时被一个元素包含的元素
七、伪类选择器：
	1. 伪类专门用来表示元素的一种特殊状态
		比如：访问过的超链接，普通的超链接，获取焦点的文本框
	2. 当需要为这些特殊状态的元素设置样式，就可以使用伪类
	3. 给链接定义样式
		①有四个伪类可以让你根据访问者与该链接的交互方式，将链接设置成4种不同的状态
		②正常链接(a:link)：没有访问过的链接
		③访问过的链接(a:visited)：浏览器通过历史记录来判断一个链接是否被访问过，
									由于涉及用户的隐私问题，所以使用a:visited只能定义字体的颜色
		④鼠标滑过的链接(a:hover)：鼠标移入状态
		⑤正在点击的链接(a:active)：超链接被点击的状态
	4. :hover和:active：不仅超链接可以使用伪类，其他元素照样可以
	5. :focus：文本框获取焦点
	6. ::selection:为选中的文本添加样式。
					注意：在火狐浏览器中需要采用另一种方式::-moz-selection
八、伪元素：使用伪元素来表示一些特殊的位置
	1. :first-letter：获取元素中的首位字符
	2. :first-line：获取元素中的第一行
	3. :before：表示元素最前边的部分，
				一般before都需要结合content这个样式一起使用，可以通过content向before和after的位置添加一些内容
				举例：p:before{content:"我是before;}
    4. :after：表示元素的最后面；举例：p:before{content:"我是before;
九、属性选择器
	1. title属性：这个属性可以给任何标签指定，当鼠标移入到元素上是，元素的title属性值将会提示文字显示
	2. 属性选择器：可以根根据元素中的属性或者属性值来选取元素。
					语法：①元素[属性名]{}:选取含有属性的元素
						  ②元素[属性名=属性值]：选取含有属性值的属性名的元素
						  ③元素[属性名^=属性值]：选取属性值中含有指定内容开始的属性名的元素
						  ④元素[属性名$=属性值]：选取属性值中含有指定内容结尾的属性名的元素
						  ⑤元素[属性名*=属性值]：选取属性值中包含指定内容的属性名的元素
十、子元素的伪类
	1. first-child：父元素的第一子元素，举例：p:first-child{},即是满足p标签，又是父元素的一个子元素
	2. last-child：父元素的最后一个子元素
	3. nth-child：可以任意选中一些子元素，举例：p:nth-child(i){},i=even表示偶数位置的子元素；i=odd表示奇数位置的子元素。
	4. first-of-type、last-of-type、nth-of-type：在当前类型的子元素中选取，而child是在所有的子元素中选取
十一、兄弟元素选择器
	1. +：可以选中一个元素紧挨着的后一个元素
	2. 举例：span+p：选中的是span后面的p元素，如果p不是紧挨在span后面，则不符合要求
	3. ~：选中一个元素后边的所有元素
	4. 举例：span~p：选中的是span后边的所有p元素，如果不是p元素，则不符合要求
十二、否定伪类
	1. 作用：可以从已选中的元素中删除某些元素
	2. 语法：:not(选择器)，举例：p:not(选择器)，表示的是选中除了符合要求的类属性值的元素之外。
十三、样式的继承
	1. 在CSS中，祖先元素上的样式，也会被他的后代继承，利用继承，可以将一些基本的样式设置给祖先元素，这样所以的后代元素将
		会继承这些样式。
	2. 并不是所以的样式都会被后代元素都继承，比如所有与背景相关的样式。
十四、选择器的优先级
	1. 当使用不同的选择器，设置同一个元素并且设置相同的样式时，设置样式之间产生了冲突
	2. 采用那个选择器设置样式，取决于选择器的优先级
	3. 优先级的规则
		内联样式优先级：1000
		id选择器优先级：100
		类和伪类的优先级：10
		元素（标签）选择器优先级：1
		通配选择器优先级优先级：0
		继承的样式没有优先级
	4. 当选择器中包含多种选择器时，会将多种选择器的优先级相加，再比较
		但是，选择器优先级计算不会超过它的数量级
	5. 如果选择器优先级一样时，选择靠后的选择器
	6. 并集选择器的优先级时单独计算
	7. 可以在样式的最后加一个!important，则此时该样式将会获得一个最高的优先级，
		其将会优先于所有的样式显示甚至超过内联样式，但是在开发中尽量避免使用!important。
	8. 涉及到a标签的伪类一共有四个：:link、:visited、:hover、:ative，这四个伪类的优先级是一致的，但是当hover位于active下面时，
		active失去点击变化的效果，原因是因为样式优先选择靠后的选择器，而当link和比visited位于hover和active后边时，hover和active失去效果
