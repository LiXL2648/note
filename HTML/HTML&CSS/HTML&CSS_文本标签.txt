<em>/<i>和<strong>/<b>
	1. <em>/<i>标签用于表示一段内容中的着重点
	2. <Strong>/<b>标签用于表示一个内容的重要性
	3. 这两个标签可以单独使用，也可以一起使用
	4. 通常<em>/<i>用于显示斜体<strong>/<b>用于显示粗体
	5. <i>和<b>没有任何语义，仅仅单独显示样式，而<em>和<strong>属于语义标签
	6. h5规范中规定，对于不需要着重的内容而是单纯的加粗或者是斜体，可以使用<i>和<b>标签用于表示一个内容的重要性
<small>
	<small>标签中的内容会比他的父亲元素小
	在h5中使用small标签来表示一些细则的内容
	比如：合同中的小字，网站上的版权声明都可以使用small
<cite>：网页中的所有加书名号的内容都可以使用cite标签，表示参考的内容，比如书名、歌曲、电影等等
<q><blockquote>：
	1. <q>表示一个短的引用（行内的引用），其内容浏览器会默认加上引号，表示引用
	2. <blockquote>表示一个上的引用（块级引用）
<sup>和<sub>:
	1. <sup>：表示上标，内容在上标的位置
	2. <sub>：表示下标，内容在下标的位置
<ins>和<del>：
	1. <ins>：下划线，表示插入的内容，自动添加下划线
	2. <del>：删除线，表示删除的内容，自动添加删除线
<code>和<pre>
	1. <pre>：预格式标签，会将代码中的格式保存，不会忽略多个空格
	2. <code>：语义标签，不会保留代码中的格式，用来表示内容是代码，两个结合使用
列表：
	1. 无序列表：
		①使用ul标签来创建无序列表
		②使用li来创建列表项，一个li就是一个列表项
		③通过type可以修改无序列表的项目符号
			A.disc，默认值，实心圆点
			B.square，实心的方块
			C.circle，空心圆
			注意：项目符号一般不适用，不同浏览器显示不同,如果需要设置项目符号，默认的项目一般不用，通过采用为项目
					符号添加背景图片
		④去掉项目符号,通过css样式去掉项目符号：list-style:none
		⑤ul和li都是块元素
	2. 有序列表：
		①有序列表与无序列表类似，使用ol代替ul
		②使用li来创建列表项，一个li就是一个列表项
		③采用type属性设置有序列表的的序号类型
			A.1，默认值，使用阿拉伯数字
			B.a/A，采用小写或者大写字母作为序号
			C.i/I，采用罗马字符作为序号
			D.ol也是块元素
	3. 列表之间都是可以互相嵌套，可以在无序列表中嵌套有序列表，也可以在有序列表中嵌套无序列表
	4. 定义了表：
		①定义列表用来对一些词汇或者内容进行定义
		③使用dl来创建一个定义列表，dl中有两个子标签dt/dd
		④dt:被定义的内容
		⑤dd:对定义内容的描述
