一. 标题标签：<h1></h1>
	1. 在HTML中，标题标签分成6级标题标签h1~h6
	2. 在显示效果上h1最大，h6最小，但是文字的大小我们并不关心
	3. 使用HTML标签时，关心的是标签的语义，我们使用的标签都是语义化标签
	4. 6级标题中，h1最重要，表示一个网页中的主要内容，h2~h6重要性依次降低
		对于搜索引擎来说，h1的重要性仅次于title，搜索引擎检索完title，会立即查看h1中的内容
		h1标非常重要，它会影响到页面在搜索引擎中的排名，页面只能写一个h1，如果存在多个，搜索引擎会将该网页视为垃圾网页
	5. 一般页面中标题标签只使用h1/h2/h3/，h3以后的基本不使用
二. 段落标签：<p></p>
	1. 段落标签用于表示内容中的一个自然段
	2. p标签中的文字，默认独占一行，并且段落与段落之间会有间距
	3. 在HTML中，字符之间写再多的空格，浏览器也当作一个空格解析
三. 换行标签<br/>：在页面中可以使用标签<br/>表示换行，换行标签是一个自结束标签
四. 水平线标签<hr/>：在页面中生成一条水平线，水平线标签是自结束标签
五. 实体：
	场景：
		在HTML中，一些如<>这种特殊字符是不能直接使用的，需要使用一些特殊的符号来表示这些特殊字符
			这些特殊符号称为实体（转义字符串）
		浏览器解析到实体时，会自动转化为相应的字符
	实体的语法：
		1. &+实体名字+;：
			①<:&lt;
			②>:&gt;
			③空格:&nbsp;
			④版权符号:&copy;
六. 图片标签
	1. 使用img标签来向网页中引入一个外部图片，img标签也是一个自结束标签
	2. 属性：
		①src:设置一个外部标签的路径
		②alt:当图片加载不了时对图片的描述/搜索引擎可以通过alt属性来识别不同的图片
			但不写alt属性时，搜索引擎不会对img中的图片进行收录
		③width：修改宽度，单位为：px
		④height：修改高度，单位为：px
			注：宽度和高度两个属性只改变一个属性时，另一个属性会等比例变化。当宽度和高度都改变时，会按照自己修改的参数进行修改
	3. 路径：
		①src属性配置的是图片的路径，目前我们所需要的路径都是都是相对路径
		②相对路径：指的是相对于当前资源所在目录的位置
		③可以使用../来返回上一级，返回几级目录就写几个../。
	4. 图片格式:
		①JPEG（JPG）：
			JPEG图片支持的颜色比较多，可以压缩，但是不支持透明
			一般使用JEPEG来保存颜色比较丰富等图片
		②GIF
			GIF支持的颜色比较少，只支持简单的透明，支持动态图
			图片颜色单一或者是动态图时可以使用GIF
		③PNG
			PNG支持的颜色多，并且支持复杂的透明
			可以用来显示颜色复杂的透明的图片
		④使用原则
			效果不一致使用好的
			效果一样使用小的
七. meta标签
	1. 使用meta标签可以设置网页的关键字,name属性写的是关键字，content写的是关键字的内容，最多写5个，之后默认不显示。
		<meta name="keywords" content="HTML5"/>
	2. 还可以用来对网页的描述,name属性写的是描述，content写的是描述的内容
		<meta name="description" content=""/>
	3. 使用meta可以用来做请求重定向<meta http-equiv="refresh" content="秒数;地址"/>
		<meta http-equiv="refresh" content="5;url=http://www.baidu.com"/>
八. HTML语法规范
	1. HTML中不区分大小写，但是一般都使用小写
	2. HTML中的注释不能嵌套
	3. HTML标签必须是完整的结构，要么成对出现，要么自结束
		浏览器会以最大的善意去解析页面，所有不符合语法规范的内容，浏览器会自动补全
	4. 自结束标签/可加可不加，有一个是XML标准，一个是H5标准
	5. 标签可以嵌套，但是不能交叉嵌套
	6. HTML标签中的属性必须有值，且值必须在引号中
九. 内联框架
	1. 使用内联框架可以引入一个外部页面
	2. 使用iframe来创建一个内联框架
	3. 属性：
		①src:指向一个外部页面的路径，可以使用相对路径
		②name:可以提供给a标签使用，以达到在内联框架中打开超链接
	4. <iframe src="相对路径"/>
	5. 在现实生活中不推荐使用，因为内联框架中的内容不会被搜索引擎所检索
十. 超链接
	1. 使用超链接可以让我们从一个页面跳转到另一个页面
	2. 属性：
		①href：指向链接跳转的目标地址，可以写一个相对地址，也可以写一个完整的地址
		②target：可以用来指定打开链接的位置
			_self：表示在当前的窗口打开链接，默认值
			_blank：表示在新的宽口打开链接
			可以设置为一个内联框架的属性值，链接会在指定的内联框架中打开
	3. 占位：创建超链接时，如果不确定跳转的链接时，可以使用#代替，当使用#时打开链接的位置仍然是当前页面
	   回到顶部：当链接地址为#号时，可以实现点击回到顶部
	4. href="#id属性值"：可以通过设置标签的id属性，使用超链接跳转到该标签的位置
	5. href="mailto:邮箱地址"（发送电子超链接）：点击超链接以后自动跳转到默认的邮箱客户端。
十一.center标签：center标签中的内容都会居中显示，不建议使用。
十二.id属性：标签的唯一标识，每个标签都可以设置，id属性每个页面只能出现一次，不能重复
十三、块元素和内联
	1. 块元素：所谓的块元素指的是会独占一行的元素，无论内容多大，也会独占一行
		分类：div、p、h1~h4.....
	2. div（块元素）：div没有任何语义，纯粹一个块元素，不会为里面的元素设置任何默认样式，主要用于布局
	3. 内联元素/行内元素：所谓的行内元素，指的是只占自身大小的元素，不会独占一行
		分类：span、a、iframe、span
	4. span（行内元素）：与div相似，没有任何语义，span标签专门来选中文字，并为其设置样式
	5. 注：
		①快元素和行内元素的区别：块元素主要用来页面布局，行内元素主要用来选中文本设置样式，
								一般情况下只使用快元素包含行内元素，并不会使用行内元素包含块元素
		②a标签：可以包含任何元素，出了他本身
		③p标签：不可以包含任何块元素
