步骤：
一、准备工作：
	1. 清除页面的默认样式
		-导入外部css文件，用于清除页面的默认样式
	2. 创建外部样式表
		-导入外部样式表，使得表现与结构完全分离
		-命名规则：page-页面名
	3. class id和文件的命名规范：
		①命名时尽量使用英文，也可以使用拼音
			-不要英文与拼音混用
		②命名格式
			A. 驼峰命名法：首字母小写，每个单词的开头大写
			B. 也可以所有字母小写，每个单词之间的用_或者-隔开
	4. 考虑兼容性问题
二、网页的设计顺序
	1. 先将页面的所有HTML代码先写完，再完成CSS，但是要求较高
	2. 设计完一部分的HTML，然后再完成该部分的CSS，推荐使用这种方式
	3. 设计网页的方式是从上到下。
	
三、选择器的选择
	1. id选择器：如果该页面单独一个，可以使用id选择器
	2. 类选择器：如果页面多个部分需要设置相同的样式，则使用类选择器
	3. 推荐使用类选择器。
四、图片裁剪
	1. 保存的格式，采用web专用格式
	2. 如果是透明背景，则采用png-24的格式保存图片，但是该方式需要考虑IE的兼容性问题
	3. 如果对于图片没有要求，则需要考虑图片大小，选择图片最小的格式保存
